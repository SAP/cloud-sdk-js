Need an example of [CAP service](../../related-projects/cap.mdx) consumed with SAP Cloud SDK! Our colleagues from [Cloud Application Lifecycle Management](../../overview/sdk-used-by.mdx#sap-cloud-application-lifecycle-management-calm) team got you covered! They put together an amazing demo of creating a locally hosted [CAP service](../../related-projects/cap.mdx) and running API calls against it with a type-safe client by SAP Cloud SDK for JavaScript. This demo will get quickly get you up to speed and make familiar with the main concepts of developing cloud extensions with Cloud SDK and CAP.
