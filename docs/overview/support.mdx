If you're stuck and can't find a solution to you problem we're there for you to help!

## Public support channels

- Create an [issue towards our GitHub repository](https://github.com/SAP/cloud-sdk/issues/new/choose)
- Use `sap-cloud-sdk` tag on [Stack Overflow](https://stackoverflow.com/questions/tagged/sap-cloud-sdk)

### How responsive are you?

- You'll usually hear from us within 1-2 day. In rare cases it can take longer because of heavy load.
- The resolution time heavily depends on the issue and can't be easily predicted.

### Who replies in support?

- Support is done by SDK developers and we take it seriously.

## Reporting an incident

- As registered SAP customers, report issues by creating incidents in [SAP Support Portal](http://help.sap.com/disclaimer?site=https://support.sap.com/home.html).
- Use **XX-S4C-SDK** component to reach out to us.
